## 2. Scope

### 2.1 In Scope

- [x] 기업 데이터 수집 (xls 읽기)
- [x] 참고문서 분석 (hwpx/hwp/pdf 읽기)
- [ ] **수행계획서 생성** (hwpx 양식 복사 → XML 텍스트 치환)
  - 과제 요약서 (2장)
  - 과제 개요 (과제명, 신청구분, 기관정보, 총괄책임자)
  - 사업 목표 및 내용 (AI 스포츠 스폰서십 서비스)
  - 추진체계 및 일정
  - 참여인력 구성
  - 기대효과 및 활용방안
- [ ] **별지서식 생성** (hwp 양식 기반)
  - [별지서식 제1호] 참여기관 참여의사 확인서
  - [별지서식 제2호] 참여인력 참여확인서
  - [별지서식 제3호] 개인정보 수집·이용·제공 동의서
  - [별지서식 제4호] 현금출자(납입) 확약서
  - [별지서식 제5호] 신청자격 적정성 확인서
  - [별지서식 제6호] 기자재 구입 및 활용계획서
- [ ] **사업비 총괄표 생성** (xlsx 양식 복사 → 데이터 주입)
  - 비목별 금액 산정 (인건비, 운영비, 여비, 업무추진비, 연구용역비, 유형자산)
  - 주관기관/참여기관별 분리
  - 정부지원금 75% / 민간부담금 25% 자동 계산
  - 연락정보 시트 작성

- 온라인 접수 시스템(kdata.or.kr) 자동 제출
- 첨부서류 (사업자등록증, 재무제표 등) 수집

## 5. Risks and Mitigation

| Risk | Impact | Likelihood | Mitigation |
|------|--------|------------|------------|
| HWP 바이너리 쓰기 불가 | High | High | 별지서식은 Markdown/텍스트 가이드로 대체 출력, 사용자가 수동 입력 |

| 접수마감(4/6) 촉박 | High | Low | 골격 먼저 생성, 세부 본문은 단계적 보완 |

## 6. Architecture Considerations

### 6.1 Project Level Selection

| Level | Characteristics | Selected |
|-------|-----------------|:--------:|
| **Starter** | Python 스크립트 기반, 단일 실행 | **O** |

### 6.2 Key Technical Decisions

| Decision | Selected | Rationale |
|----------|----------|-----------|
| 언어 | Python 3.14 | 이미 설치됨, 파일 처리 라이브러리 풍부 |
| HWPX 읽기/쓰기 | zipfile + xml.etree | 표준 라이브러리, HWPX는 ZIP/XML |
| HWP 읽기 | olefile (PrvText) | pyhwp 설치됨, OLE 스트림 접근 |
| HWP 쓰기 | Markdown 가이드 출력 | 바이너리 HWP 직접 생성 불가 |
| XLSX 읽기/쓰기 | openpyxl | 설치 완료, 수식 유지 가능 |
| XLS 읽기 | xlrd | 설치 완료 |
| PDF 읽기 | PyMuPDF (fitz) | 설치 완료 |

수행계획서 양식.hwpx  ──┐        ┌─ generate_proposal.py ──┐        02_수행계획서.hwpx

## 9. 과제 내용 상세 (AI 스포츠 스폰서십 인텔리전스)

### 9.1 서비스 개요

**과제명**: AI 기반 스포츠 스폰서십 데이터 연계·분석 서비스

**서비스 설명**:
스포츠 기업(구단, 리그, 에이전시, 스폰서 기업)이 보유한 스폰서십 계약, 미디어 노출, 관중/팬 데이터를 동의 기반으로 연계하고, AI가 이를 통합 분석하여 스폰서십 가치를 실시간 측정·예측하며 기업 간 최적 매칭을 추천하는 플랫폼.

### 9.2 서비스 구성

| 구분 | 내용 |
|------|------|
| **데이터 수집·연계** | 스폰서십 계약 데이터, 미디어 노출 데이터, 관중/팬 행동 데이터, SNS 반응 데이터 |
| **AI 분석 엔진** | LLM 기반 미디어 자동 분석, 스폰서십 가치 예측 모델, 기업-종목 매칭 추천 엔진 |
| **실증 서비스** | 스폰서십 ROI 대시보드, 실시간 미디어 가치 측정, AI 매칭 추천, 효과 비교 분석 |
| **데이터 전송 체계** | 기업 동의 기반 API 연계, 표준 데이터 포맷 정의, 전송 이력 관리 |

### 9.3 AI 기술 적용

| AI 기술 | 적용 영역 | 기대효과 |
|---------|----------|----------|
| **자연어처리(NLP/LLM)** | 미디어 기사·SNS 자동 분석, 스폰서십 노출 자동 인식 | 분석 시간 90% 단축 |
| **예측 모델링** | 스폰서십 ROI 예측, 시장 가치 변동 예측 | 사전 의사결정 지원 |
| **추천 알고리즘** | 기업-스포츠 종목 최적 매칭, 예산 최적 배분 추천 | 매칭 정확도 향상 |
